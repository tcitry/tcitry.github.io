                                  一些定律

                                  摩尔定律#

                                  摩尔定律是英特尔创始人之一戈登·摩尔的经验之谈，其核心内容为：集成电路上可以容纳的晶体管数目在大约每经过18个月便会增加一倍。

                                  康威定律#

                                  计算机科学家 马尔文·康威 1976 年提出，设计系统的架构受制于产生这些设计的组织的沟通结构。

                                  墨菲定律#

                                  墨菲定律（Murphy’s Law）的原句是：如果有两种或两种以上的方式去做某件事情，而其中一种选择方式将导致灾难，则必定有人会做出这种选择。一般会说 “if it can go wrong, it will.”（任何可能出错的地方终将出错）。

                                  帕金森定律#

                                  英国作家西里尔·诺斯古德·帕金森在 1954 年写的一本书中介绍了一个观点，认为给一个项目多少时间，它都能将之耗完。（wiki：在工作能够完成的时限内，工作量会一直增加，直到所有可用时间都被填充为止）。

                                  马太效应#

                                  马太效应（Matthew Effect）为强者愈强弱者越弱之现象，即是贫者愈贫富者愈富。现实中就是指，只要获得了每一点的成功，就会产生累积优势，使之拥有更大的机会获得更卓越的成就。也是造就了贫富差距的原因之一。

                                  边际成本#

                                  边际成本（Marginal Cost），亦作增量成本（Incremental Cost），在经济学和金融学中指的是每增产一单位的产品或多购买一单位的产品所增加的成本。这个概念表明每一单位的产品的成本与总产品量有关。比如，仅生产一辆汽车的成本是极其巨大的，而生产第一万辆汽车的时候成本就会低得多，而生产第一百万辆汽车的时候已经形成产业链，成本就更低了。还有，比如一款应用软件的边际成本几乎为零，因为软件只需要拷贝一下就可以增加一份。

                                  普瑞玛法则#

                                  普瑞马法则是由美国心理学家大卫·普瑞马(David Premack)在20 世纪60 年代提出的一种行为原理。 该法则指出，更有可能发生的行为可以用来强化不太可能发生的行为。 普瑞马法则的基本原理是，通过强化一个容易发生的行为(称为’高概率行为”)来增加另一个不太可能发生的行为(称为“低概率行为”)的发生频率。

                                  沉没成本#

                                  沉默成本（又称沉没成本）是经济学和心理学交叉领域的重要概念，指已经发生且无法通过任何决策回收的支出，包括时间、金钱、精力等资源。其核心特点是不可逆性与无关性——即这些成本不应影响当前或未来的决策，而应专注于边际成本和未来收益。

                                  一、沉默成本的本质与特点

                                  1. 不可回收性
                                    沉默成本一旦投入便无法收回，如已购买的设备折旧、已支付的会员费、已投入的研发资金等。这类成本无论后续决策如何选择（如继续或终止项目），都无法改变其“沉没”属性。

                                  2. 决策无关性
                                    理性决策应基于未来潜在收益与新增成本，而非过去的投入。例如，电影看到一半发现难看时，票钱已是沉默成本，继续观看的决策应取决于剩余时间的愉悦度，而非“不想浪费票钱”的心理。

                                  3. 心理绑架效应
                                    人类对损失的厌恶感是收益的两倍（损失厌恶心理），导致人们倾向于通过追加投入试图“弥补”沉默成本，形成“沉没成本谬误”。典型案例包括：

                                    • 诺基亚固守塞班系统：因不舍前期巨额投入，错失智能机时代，市场份额暴跌；
                                    • 协和客机项目：英法政府明知无效益，仍追加8倍预算，最终沦为亏损的“科技奢侈品”。

                                  二、沉默成本与机会成本的区别
                                  沉默成本强调“已发生且不可逆”，而机会成本指因选择某一选项而放弃的其他潜在收益。例如：

                                  • 100万元存入银行（年利率3.5%）的机会成本是放弃投资年收益10%的10万元利润；
                                  • 选择玩过山车的机会成本是失去溜冰的体验。

                                  三、沉默成本的影响场景

                                  1. 商业决策
                                    企业常因沉默成本陷入非理性扩张，如威马汽车前期重资产投入导致转型困难。正确做法是建立“亏损红线”，定期评估项目ROI（投资回报率）。

                                  2. 个人生活

                                    • 消费行为：购买高价商品后因VIP资格不愿承认错误选择（如奢侈品购物陷阱）；
                                    • 情感关系：因长期投入不愿结束不健康的关系，形成情感勒索。

                                  3. 社会机制
                                    商业设计中常利用沉默成本心理，例如：

                                    • 健身房捆绑年卡与私教课，利用已付年费绑定用户；
                                    • P2P平台通过“投资券”诱导持续投入，制造锁死效应。

                                  四、如何规避沉默成本陷阱

                                  1. 设定止损点
                                    明确“退出阈值”，如观影30分钟难看即离场，网购商品7天无理由退货。

                                  2. 未来导向思维
                                    决策时问：“如果现在零投入，我还会选择继续吗？”聚焦边际收益而非总成本。

                                  3. 接受“断舍离”
                                    企业案例：英特尔果断放弃Timna芯片避免更大损失；个人案例：及时结束无益社交或投资。

                                  4. 外部视角介入
                                    通过第三方评估（如顾问、数据分析工具）剥离情感因素，客观判断项目价值。

                                  五、沉默成本的特殊价值

                                  少数场景下，沉默成本可能具有战略意义：

                                  • 情感修复：婚姻中因共同回忆选择努力修复关系，可能创造新价值；
                                  • 品牌建设：长期投入塑造品牌形象，虽短期难量化收益，但具备长远价值。

                                  结语
                                  沉默成本如同“过期的船票”，执着于它只会阻碍前行。理性决策的本质是与过去和解，将资源分配给能创造未来的选择。正如经济学提醒的：“不要为打翻的牛奶哭泣，除非你能用它浇灌新的花朵。”
