            ButtonStyle 是 SwiftUI 提供的一种自定义按钮外观和交互方式的机制，允许我们创建可复用的按钮样式，并在多个 Button 组件上应用。

            🔹 基本使用 #

            我们可以创建一个自定义的 ButtonStyle，然后在多个 Button 组件上使用它。

            📝 自定义 ButtonStyle #

            struct CustomButtonStyle: ButtonStyle {
    func makeBody(configuration: Configuration) -> some View {
        configuration.label
            .padding()
            .background(configuration.isPressed ? Color.gray : Color.blue)
            .foregroundColor(.white)
            .cornerRadius(10)
            .scaleEffect(configuration.isPressed ? 0.95 : 1.0) // 点击时缩小
            .animation(.easeInOut(duration: 0.2), value: configuration.isPressed)
    }
}

            📌 关键点：

            • configuration.label 代表 Button 的内容（Text、Image 等）。
            • configuration.isPressed 表示按钮是否被按下。
            • .background() 和 .scaleEffect() 让按钮在点击时有视觉反馈。
            • animation(.easeInOut(duration: 0.2)) 让交互更平滑。

            🎯 使用 ButtonStyle #

            Button("Tap Me") {
    print("Button tapped!")
}
.buttonStyle(CustomButtonStyle()) // 应用样式

            🔹 预设 ButtonStyle #

            SwiftUI 也提供了一些内置样式，可以直接使用：

            Button("Default") { }
    .buttonStyle(DefaultButtonStyle()) // 默认样式

Button("Plain") { }
    .buttonStyle(PlainButtonStyle()) // 去掉所有默认样式

Button("Borderless") { }
    .buttonStyle(BorderlessButtonStyle()) // 类似 Plain，但不会影响菜单按钮

            🔹 适用于特定类型的按钮 #

            如果你的 ButtonStyle 只适用于某种特定的 Button，可以使用条件限制：

            struct RedButtonStyle: ButtonStyle {
    func makeBody(configuration: Configuration) -> some View {
        configuration.label
            .padding()
            .background(Color.red)
            .foregroundColor(.white)
            .cornerRadius(8)
    }
}

// 只让 `Destructive` 类型的按钮使用这个样式
extension ButtonStyle where Self == RedButtonStyle {
    static var destructive: RedButtonStyle { RedButtonStyle() }
}

Button("Delete") { }
    .buttonStyle(.destructive) // 使用自定义扩展

            🔹 ButtonStyle vs .buttonStyle() vs .background() #

            方式适用场景可复用性
            .background() + .padding()适用于单个按钮需要手动复制样式
            .buttonStyle(CustomButtonStyle())适用于多个按钮高，可复用
            .buttonStyle(DefaultButtonStyle())适用于默认按钮低
            .buttonStyle(PlainButtonStyle())适用于去掉默认交互低

            🚀 如果要自定义多个按钮，使用 ButtonStyle 更好！

    static var previews: some View {

            3. 完全自定义按钮组件 #

            封装可复用的按钮组件 #

            struct ThemedButton<V: View>: View {
    let title: String?
    let icon: Image?
    let action: () -> Void
    let buttonType: ButtonStyle {
        didSet {
            // 根据按钮类型自动调整样式
            switch buttonType {
            case .primary:
                return .borderedProminent
            case .secondary:
                return .bordered
            case .text:
                return .borderless
            }
        }
    }
    
    init(
        title: String? = nil,
        icon: Image? = nil,
        action: @escaping () -> Void,
        buttonType: ButtonStyle = .primary
    ) {
        self.title = title
        self.icon = icon
        self.action = action
        self.buttonType = buttonType
    }

    var body: some View {
        Button(action: action) {
            HStack(alignment: .center, spacing: 8) {
                icon?.resizable().aspectRatio(contentMode: .fit).frame(width: 24)
                
                if let title = title {
                    Text(title)
                        .font(.subheadline)
                        .fontWeight(.semibold)
                }
            }
            .buttonStyle(buttonType)
            .tint(.white) // 统一文字颜色
            .padding()
            .cornerRadius(12)
        }
    }
}

            4. 状态驱动样式（根据按钮状态改变外观） #

            实现按压效果的动态按钮 #

            struct DynamicButton_Previews: PreviewProvider {
    static var previews: some View {
        Button(action: {}) {
            ZStack {
                // 动态背景
                Rectangle()
                    .fill(Color.blue.opacity(0.5))
                    .scaleEffect(isPressed ? 0.95 : 1)

                // 文字内容
                Text("Press Me")
                    .font(.title2)

            .frame(maxWidth: .infinity)
            .padding()
            .cornerRadius(12)
        }
        .previewStates(["Normal", "Pressed"]) // 预览不同状态
    }
}

            6. 进度指示按钮 #

            实现带加载状态的按钮 #

            struct LoadingButton_Previews: PreviewProvider {
    @State private var isLoading = false
    
    static var previews: some View {
        Button(action: {
            isLoading = true
            // 模拟异步操作
            DispatchQueue.main.asyncAfter(deadline: .now() + 2) {
                isLoading = false
            }
        }) {
            if isLoading {
                ProgressView()
                    .tint(.white)
                    .frame(width: 24, height: 24)
            } else {
                Text("Submit")
                    .font(.title2)
                    .fontWeight(.semibold)
                    .foregroundColor(.blue)
            }
        }
        .buttonStyle(.borderless)
        .padding()
        .cornerRadius(12)
    }
}

            7. 完整按钮模板（含按下动画） #

            高级按钮实现 #

            struct AdvancedButton_Previews: PreviewProvider {
    static var previews: some View {
        Button(action: {}) {
            ZStack {
                // 背景动画层
                Circle()
                    .fill(Color.blue.opacity(0.2))
                    .scaleEffect(1.1)
                    .opacity(0.5)
                    .animation(.easeInOut(duration: 0.3))

                // 主内容
                VStack(alignment: .center, spacing: 6) {
                    Image(systemName: "arrow.upward")
                        .foregroundColor(.white)
                        .font(.system(size: 28))
                        .rotationEffect(.degrees(isPressed ? -10 : 0))
                    
                    Text("Upload")
                        .font(.subheadline)
                        .fontWeight(.semibold)
                        .foregroundColor(.white)
                        .transform(.translateY(isPressed ? -2 : 0))
                }
            }
            .frame(maxWidth: .infinity)
            .padding()
            .cornerRadius(12)
            .transition(.all, duration: 0.3)
        }
    }
}

            关键技巧总结 #

            1. 按钮类型控制： • .buttonStyle(.borderedProminent)：iOS 原生样式 • .buttonStyle(.bordered)：带边框样式 • .buttonStyle(.borderless)：无边框文本按钮

            2. 状态管理： • 使用 @State 监听按钮按下状态 • 通过 .previewStates 预览不同状态

            3. 性能优化： • 对复杂图形使用 LazyVStack/LazyHStack • 避免在按钮内使用 heavy computations

            4. 平台适配：

              struct YourButton_Previews: PreviewProvider {
    static var previews: some View {
        YourButton()
            ..previewDevice("iPhone 13 Pro Pro") // 指定设备预览
    }
}
