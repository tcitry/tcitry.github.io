    我的 2021 年总结 Recommended

    12月 31, 2021
    Life, Recommended, 年度总结

    这是一篇 2021 年的年终总结，如果给 2021 一个关键词：那绝对是“价值观”。因为 2021 这一年是我价值观发生巨大改变的一年。回头看我还是感谢这一年发生过的事情，生活的很多道理我们都懂，但是如果没有经历过可能就没有深入的思考和实际的改变。

    故事 #

    故事的起因从一件事情说起，2021年 1 月初，发年终奖的前夕，产研部门的领导突然开会说上年度的年终奖由于部门绩效原因，绩效为 B 及以下的同学的年终奖不发了，虽说是部门的绩效原因，但是 B 也不发这种操作就很奇怪，而且公司也没有什么重大经济损失。因为我司即使绩效为 A 也就发一个月的工资，加上 A 的人很少，所以我暂时保持了冷静，觉得钱也不多，就是年底没有年终奖跟家人朋友说起来挺丢人的。那么这件事违法吗？好像不违法，因为劳动合同就是 12 个月的薪资，所以也不需要去反抗或投诉，即使违法，投诉也是成本很高的事情，万不得已也不值得一试。

    公司不发年终奖这事，让我觉得是我的工作付出没有得到相应的回报，事实上，在 2020 年下半年，我主导的一个项目已经成功给部门解决了很大的问题，这个是大家都能看得到的，并且这个项目我也确实加班加点投入了很大的精力，当然也学到了很多东西，后来自己也成功转型到了 Go 的技术栈。在后来的两个月里，这件事我也是越想越郁闷，而且过完年的我瘦了十斤，要知道往常过年一般能长十斤，希望我妈看不到这里，太不孝了。随后过完年公司又走了一批同事，包括我的直属领导，再之后我认为就没有与此相关的事情了，不再多说。

    价值观 #

    那么现在到年底了，为什么我仍然还在这家公司？按说我应该早就准备面试然后离职啊。我觉得这其中最大的原因可能在于我之前没有想清楚一个问题：下一份工作要做什么才能避免以后还会发生同样的事情。所以这件事其实引发了我更深层次的思考，而且前后持续了一年的时间，直到2021年底，我才确认我开始逐渐捋清楚了面对这种问题时，身为“打工人”的我应该怎么办。

    这里我先列一下我最初面临的问题：

    1. 新的工作辛苦付出一年后，公司同样可以说今年不发年终奖了。
    2. 我非顶尖的技术人才，需要一定时间的准备和面试才能换一份新的工作。
    3. 我的岗位是业务很重的后端，工作中的绝大部分精力会花费在公司业务流程和数据准确性上面，简单说就是为公司想的太多，为自己想的太少。

    就这三个问题我花了一年来复盘：其中问题 3 导致了问题 2，问题 2 进而让我遇到问题 1 时束手无策。特别是问题 3，我相信只从我的工作内容上讲带给公司的收益肯定是正向的，但是带给我自己的收益是什么呢？是提交在内部 wiki 上的文档和 n 多微服务间调用的流程图吗？是实现流程图的 CURD 和 HTTP 请求的代码吗？是写了无数用来处理临时紧急需求、或是修数的脚本吗？是跨部门争论了很多遍的“自创概念”吗？是几乎每天都要花时间排查的又是谁离职后交接给我的某个系统的 Bug 吗？显然这些占据了我工作中大部分时间的工作内容都不是我自己的收益，但是解决这些问题确实给公司带来了价值。

    为了避免有人抬杠，另外说一句，工作中剩下的内容还是有收益的，比如当我们意识到问题严重时，我们就会新开一个“服务”或者“概念”或者“流程”，来解决遇到的问题，可能我们就会用更通用更先进更适合当下的模型解决它，尤其当我们遇到高并发问题时会异常的兴奋，因为终于找到了一项面试时可以聊的工作内容。

    三个问题 #

    就这样工作上我陷入了一个地狱级的恶性循环，像深陷在沼泽地一样出不来。但是问题还得要解决不是么，现在我把我面临的这三个问题再抽象一下，就是：

    1. 如何让工作中的 付出 == 回报？
    2. 如何提高自己的市场竞争力？
    3. 如何让工作的内容对我本身更有价值？

    到这里我要引用一篇博客里的一段话：

    有的读者可能会说我有一份 996 的工作，没有这么长的业余时间，那么我们应该利用一切业余时间满足技术的提升并换一个工作时间合理、不存在无意义加班的工作。在长时间工作且没有业余时间提升自己的情况下，我们很难实现个人的成长和技术的进步，工程师需要在业余时间来提升自己的技术、了解更多的知识。在这个行业工作，我们很难遇到完全不加班的工作，但是无意义的加班以及每天 12 小时的工作永远都是要拒绝的。

    在这段话里，作者的核心理念是：尽量避免无意义的加班工作并且注重技术人自身的成长。这份工作曾经对我很有意义，我甚至通过主动加班来达成更高的成果，但是随着人员流失、每天的问题排查、手工处理数据、紧急需求以及付出 != 回报的若干稻草下，我得出了这份工作如今对我没有意义的结论。

    业余时间 #

    除了正常工作以外，接下来讲讲我这一年的业余时间里做了什么。

    面试刷题 #

    这个是肯定的，但是由于 3 年没有面试，好多算法题已经生疏了，这方面处于断断续续捡起来的状态。另外出于对工作性质的认识上的改变，我开始质疑自己要不要继续找一份纯 Golang 后端的职位，所以在其他方向的准备上，有点慌不择路，不知道着重看什么内容，但肯定还是开发岗。

    量化交易 #

    本来在焦虑着准备面试找工作，但是情况在四月一个周五的晚上接到一个老家好友打到凌晨两点的语音电话开始发生了改变。后来的几个周末里，老乡和我，一个人提供交易策略，一个人写代码实现，最终在 FMZ 平台跑了起来，通过飞书机器人实现了实时的交易通知，并且可以查看总体盈亏情况。可惜最终的收益并不理想，毕竟是人人都炒的币圈，但是我学到了一些交易策略，比如海龟、网格、资金费套利等，还有一些数据指标，对在股市做一名合格的韭菜还是很有帮助的。

    远程兼职 #

    后来一个北京的朋友找我，问我有个项目要不要一起做，远程办公，和另外几个朋友在一起，就有了重写后的这个项目，我就顺便体验了一下周末远程办公的感受。

    远程工作的优点我就不说了，每个人都有各自的需求。我来列举下发现的缺点：

    1. 都是兼职的话时间肯定不固定，这样会直接导致沟通不便，如果分工上还有相互依赖的情况，项目进度会明显下降，这对团队协作也是挑战。
    2. 远程工作的人没有不摸鱼的吧，肯定会东看看西看看，虽然涉及的面更广了，但是不聚焦。如果自己解决问题的目标感不够强的话，虽然自由了，但是时间可能更不容易用到刀刃上，这样对自己也没有好处。

    独立软件 #

    工作需要，今年开始学习前端的框架和技术，另外也打算开发一个采用前端技术的桌面软件来练手，就在业余时间也学习了 React + Electron + TypeScript，只能说前端的技术栈太多，一时间完全学不完，做每个功能基本都会遇到问题，然后顺着问题从基础知识再开始学，可能在业余时间里一个月才能实现一个小功能，进度非常慢，现在已经有了一个初步的模型，后续会继续完善争取早日发布。

    另外出于对桌面软件（主要是 macOS）的技术栈的好奇，也学习了 Swift，并过了一遍官网的 SwiftUI 教程并实现了其 demo，其实大学自学过 Objective-C 和那时的 Swift，不过最后实习还是找了后端岗位，可能因为觉得后端更“核心”一点吧，现在我也非常看好 macOS 平台原生软件的开发，如果遇到好的想法应该会尝试一下。

    自驾旅行 ✈️ #

    主要也是疫情不敢出去玩，出去了也怕回不来，只能见缝插针，九月份租车跑了一趟张北草原天路，沿途风景的体验还是非常好的，就是周边吃住的环境整体都比较差吧，基本都是农家乐。

    十一陪媳妇回大连，又自驾了一圈，不得不说现在大连真是干净美丽的城市，比毕业的时候感受更加强烈，空气湿润清新，特别适合养生，加上房价不贵，教育在辽宁也属于最好的地区之一，而且辽宁的高考分数线本身比较低，很适合以后回来生活。有人说大连经济不行了，但是我觉得经济不好并非不适合生活，反而人少了更能图个清净，不用排队挤地铁也挺好的，还有重要的一点是辽宁的民间乒乓球氛围是真的好，球馆贼多。

    不自觉的，今年开始在 B 站关注一些旅行博主和汽车自媒体，比如“杨旭游记”、“房车情报”、“几分钟义务教室”、“极速拍档”，差不多都是今年开始关注的，确实对这些方面开始感兴趣了。

    继续打球 🏓️ #

    乒乓球只要有条件就一直没有断，上半年几乎每个周末没有疫情时都会打一下午，夏季 6、7 月份趁疫情好转还与球友参加了比赛，后来 10 月以后疫情原因只能又停止了训练。对了，这一年还有一个球友辞掉了北京的技术岗位回老家开了个球馆，实名羡慕。

    继续练琴 🎹 #

    今年练习了《天空之城》《卡农》《致爱丽丝》的初级版谱子，不过最近换眼镜导致眼睛难受没有坚持练习，可能又忘了。

    继续阅读 📖 #

    年初时买了一部阅读器，非常轻便，上班通勤必备。另外搬家前卖了一大批纸质书，只留下几本经典的大部头，其余全面转向了电子书，可以通过这个油猴脚本在豆瓣上查找在阅读平台有版权的书，找书完全不是问题。目前关注的基本偏向技术类、历史类、小说类，因为读了内容也不一定是我的，消化和感悟能力有限，具体心得体会就不说了，很多时候知道了有那么回事，用到的时候再反复回去读。

    个人体感，只通过微信阅读的排行榜来看，每周阅读超过 5 个小时的人就很少了。2021 年我一共在微信阅读上面花了 167 个小时，有一些用到的书微信阅读上没有，可能会去得到电子书上找，总的时间大致估计应该在 200 个小时以上，平均到每周差不多也就是 5 个小时，关于阅读后的输出几乎都能从我的博客里找到。

    通过这一年“细水长流”似的强度来阅读，也了解了下自己的阅读水平，现在看到一本书时，基本上已经可以判断自己多久可以看完，一次阅读可以达到什么效果，这样时间安排上会更加从容。比如有的技术类的入门实践的书一两晚的加班就可以看完，有的小说比如《两京十五日》每天地铁通勤的时候看花了两周，而《挪威的森林》这本书我一周可以看两遍，有的历史类的《万历十五年》读第一天就发现这本书可能一遍不够我看懂了，然后最近当我拿起老舍的《四世同堂》的时候，我就知道这本书如果我只在每天的地铁上看可能需要花一个月，书是好书，可是手已经抖了。

    说起老舍，地铁公益广告里有看到他的一句话：生活是种律动，须有光有影，有左有右，有晴有雨，滋味就含在这变而不猛的曲折里。每天下班我都能看到这句话，每天都会品一遍。

    易经八卦 #

    新技能 Get ✓，学会了用三个硬币算卦，给朋友算好像还挺准的哈，可能是因为易经本来就是一些很重要的做人做事的道理总结，放在很多时候都管用，当然算卦是次要的，主要是易经的内容确实非常有深度，我也越来越觉得这是中华文化的根源，可以继续多花时间研究研究。

    双拼输入法 #

    新技能 Get ✓，之前已经写了一篇推荐，如果你会因打字时按太多的键而感到焦虑，这是一个解决方案，再次推荐给你。

    再谈价值观 #

    不知不觉一年过去了，我发现业余时间做的事情，可能正是我潜意识里期望、向往的事情，也许正是能带给我源源动力继续生活学习、提高自己的另一蹊径了，至于工作，一定要想清楚是公司在拿钱换你的时间，还是你在拿时间换公司的钱，这两者主被动关系是不一样的。如果想要付出多余的精力投入工作以期望得到更多回报，就一定要找一个靠谱的公司和适合自己的工作内容。现在，接前面我遇到的 三个问题，进而再重新思考：我的价值观是什么？一位前辈跟我说过：“人的价值在于工作”，这句话我是非常认同的，只不过变数也就在于工作的各个方面了，那到底做什么工作才能体现自我价值呢？我已经有了答案。

    只要想明白了，不用管别人说的对与错，自己的路自己走下去就是了。这段时间也看了 GitHub 上面 博主们的 2021 年总结，很有感触，每个人都有困境，但都在路上，山长路远，唯有珍惜时间。
